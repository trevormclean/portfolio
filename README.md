# Interactive Portfolio

A personal portfolio website featuring an **interactive graph-based interface** for exploring projects and their technical details.

Projects are represented as nodes connected to a central hub. Selecting a node reveals deeper information about the project, including challenges, technical decisions, and lessons learned.

## Live Demo

trevormclean.dev


## Features

* **Interactive project graph**

  * Projects appear as nodes connected by dynamically routed edges
  * Selecting a node reveals detailed project information

* **Dynamic SVG edge routing**

  * Edges automatically adjust their paths based on node positions
  * Prevents overlap while maintaining clean visual spacing

* **Component-based UI**

  * Modular React components for nodes, edges, and the detail panel

## Tech Stack

* **Next.js**
* **React**
* **TypeScript**
* **Tailwind CSS**
* **SVG** for dynamic edge rendering
